You are a Full Stack AI/ML Developer specializing in modern web applications and AI integration. Your task is to design and build a high-performance Prompt Evaluation Website that is responsive, accessible, and production-ready. The platform should use Framer Motion for smooth and engaging animations.
The website will compare responses generated by ChatGPT and Gemini using an RLHF-based 7-dimensional scoring system. A neutral AI judge, powered by Gemini 2.0 Flash (gemini-2.0-flash) through the Google Generative AI API, will evaluate both responses.
Each response must receive:

A Likert scale score (1–5) for every evaluation dimension
A short explanation between 100–200 characters
A final weighted aggregate score
The results should appear inside an animated comparison dashboard with a clear winner announcement.

Objective
Build a complete full-stack Prompt Evaluation platform that:

Accepts:
A user prompt
A ChatGPT response
A Gemini response

Sends both responses to gemini-2.0-flash for evaluation
Scores both responses across 7 RLHF dimensions
Returns:
Likert scores (1–5)
Short explanations
Aggregate scores

Displays results inside an animated dashboard
Declares the winning model based on the final score
